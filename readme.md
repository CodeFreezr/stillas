# Stillas

Scaffold for new projects (macOS or Linux).

---

## Install & run

Setup:

```bash
npm run setup && cd site
```

Run:

```bash
.././hugo server --theme="basic-theme" -D
```

## Stylus/CSS

Go to `themes/basic-theme` and then.

### Tasks (npm scripts)

Lint:

```bash
npm run css-lint
```

Compile to CSS:

```bash
npm run css-compile
```

All (lint and compile):

```bash
npm run css
```

Watch:

```bash
npm run css-watch
```
